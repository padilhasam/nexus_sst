# Layout aprovado aplicado

Este pacote contém ajuste de código focado somente no visual do mockup aprovado:

- `app/views/templates/header.php` e `footer.php` refeitos para o shell com sidebar/topbar.
- `public/css/layout-aprovado.css` com o design system do layout da imagem.
- Views principais ajustadas: Dashboard, Agenda, Visitas Técnicas, Check-list e GHE/Riscos.

Não foram alteradas regras de negócio, controllers ou banco de dados.
